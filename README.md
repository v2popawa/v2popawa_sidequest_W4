## Description

To win the game the user needs to reach the last plaform to win, in order to win the user needs to aviod the
falling objects. If the user gets hit it restarts, if the user reaches the last platform they win.
To move the blob the user can use their keyboard or can use the "A" and "D" key to move left and right and
the space bar to jump. Use the "N" key to restart at any time. Once they win they can click on the restart
button to play again.
